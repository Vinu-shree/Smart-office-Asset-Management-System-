# Smart Office Asset Management and Security System

IoT-based Smart Office Asset Management and Security System using RFID, Raspberry Pi Pico, Adafruit IO Dashboard, Telegram Alerts, and Webcam Verification.


## 📌 Project Overview

This project is designed to improve office/lab security and asset monitoring using Embedded Systems and IoT technologies.

The system authenticates employees using RFID cards, tracks office assets, monitors room occupancy, captures webcam verification images, and sends real-time alerts through Telegram and Adafruit IO Dashboard.


# 🚀 Features

## ✅ RFID Attendance System
- Each employee/user has a unique RFID card.
- Attendance is automatically marked when the card is tapped.
- Entry and exit timings are recorded.
- Live occupancy count is updated.


## 📷 Webcam Verification
- Webcam captures image when RFID card is scanned.
- Helps verify the identity of the person.
- Captured images are sent through Telegram alerts.


## 📦 Asset Tracking System
- Office assets like laptops, projectors, and tool kits use RFID tags.
- System tracks:
  - Who accessed the asset
  - When it was used
  - Asset activity logs
  

## 🚨 Unauthorized Asset Removal Detection
- Detects unauthorized asset movement/removal.
- Activates buzzer alarm.
- Sends security alerts through Telegram.
- Displays alerts on IoT dashboard.


## 👥 Live Occupancy Monitoring
- Counts number of people inside the office/lab.
- Displays room occupancy in real-time.
- Helps monitor room availability.


## 📊 Monthly Analytics
The system can be extended for:
- Most used asset
- Most active employee
- Total attendance tracking
- Peak usage days


## 📱 Telegram Alert System
Real-time alerts are sent for:
- Unauthorized access attempts
- Asset usage activity
- Unauthorized asset removal
- Attendance events
- Webcam verification images


## ☁️ Adafruit IO Dashboard
Cloud dashboard displays:
- Current user
- Entry/Exit status
- Asset activity
- Occupancy count
- Security alerts
- Latest activity logs
